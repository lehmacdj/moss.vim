# moss.vim

Vim/Neovim syntax highlighting for
[Moss](https://github.com/moss-lang/moss).

## Installation

With [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'lehmacdj/moss.vim'
```

## Disclaimer

This plugin was ported from the
[VS Code extension](https://github.com/moss-lang/moss/tree/main/packages/moss-vscode)
using Claude.
